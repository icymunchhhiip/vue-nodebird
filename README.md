# NodeBird SNS
Vue.js + Nuxt
* Vue.js@2 - Nuxt@2
* MySQL@5.7

# Development Settings

## Install node
```
brew update
brew install node
```

## Init
```
cd ch1/front
npm init
```
* package name: vue-nodebird-front
* 나머지 default

```
npm i vue nuxt
```

# What is Nuxt ?
* vue 기반 웹사이트를 위한 프레임워크
* 컴포넌트보다는 page 중심 (폴더 이름을 꼭 `pages`라고 써야함!!)
* pages 안을 자동으로 페이지로 만들어줌

# Run
```
npm run dev
```
